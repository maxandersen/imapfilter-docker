# imapfilter-docker

CentOS based image with imapfilter configured and setup to run your volume mounted imapfilter config files.

# Usage

`docker run -d -v ~/.imapfilter:/imapfilter/.imapfilter  maxandersen/imapfilter-docker <password>`
